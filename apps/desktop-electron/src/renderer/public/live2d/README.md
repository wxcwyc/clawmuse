Place renderer-served Live2D model assets here.

`assets://<character-id>/...` resolves to `/live2d/<character-id>/...`.

Example:

- `assets://builtin-hiyori/Hiyori.model3.json`
- `/live2d/builtin-hiyori/Hiyori.model3.json`

Do not commit third-party Live2D models here unless their asset rights are verified for this project.
