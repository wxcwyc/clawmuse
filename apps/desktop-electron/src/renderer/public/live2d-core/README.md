This directory contains the Live2D Cubism Core runtime file used by the
renderer:

`live2dcubismcore.min.js`

The renderer HTML preloads `/live2d-core/live2dcubismcore.min.js` before the Vue app boots.

The checked-in file should only be replaced with an official Live2D Cubism Core
build that is listed as redistributable by Live2D. Keep Live2D license notices
intact when updating it.

Companion license files copied from the official SDK package:

- `LICENSE.md`
- `RedistributableFiles.txt`
